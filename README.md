# Heaven Farm - Nature's Freshness Straight to Your Kitchen

Heaven Farm is a premium supplier of fresh vegetables and fruits, serving the needs of restaurants, hotels, and families in Indonesia. Based in Pangalengan, Bandung, we are committed to providing high-quality agricultural products that are pesticide-free and harvested every morning directly from the farm.

## 🌟 Key Features

- **Daily Harvest:** Products are picked every morning to ensure maximum nutrition and freshness upon delivery.
- **Pesticide-Free:** 100% organic and safe for consumption with strict quality control.
- **Business Services:** Competitive pricing and high-volume supply for restaurants, hotels, and catering partners.
- **WhatsApp Integration:** Easy access for consultation and quick ordering via WhatsApp.
- **Product Catalog:** Filter categories (Vegetables, Fruits, Cooking Kits) to simplify your search.

## 🏗️ Project Structure

This project consists of two main parts:

1.  **Static Landing Page (Root):** The original version built using HTML5, CSS3, and Vanilla JavaScript.
2.  **Next.js Application (`heaven-farm-next/`):** A modern version currently in the migration/development phase using the Next.js 14 framework.

### Folder Directory
```text
.
├── assets/             # Images, icons, and logos
├── components/         # Reusable HTML components (Static)
├── css/                # Stylesheets (styles, components, responsive)
├── js/                 # JavaScript logic (main.js)
├── heaven-farm-next/   # Next.js Application (Modern)
│   ├── app/            # App Router (Pages, Layouts, CSS)
│   ├── components/     # React Components
│   └── public/         # Static assets for Next.js
├── index.html          # Main static site page
└── setup_next.js       # Initialization script for Next.js
```

## 💻 How to Run

### Static Version
Simply open the `index.html` file in your preferred browser or use the Live Server extension in VS Code.

### Next.js Version
1. Navigate to the directory:
   ```bash
   cd heaven-farm-next
   ```
2. Install dependencies:
   ```bash
   npm install
   ```
3. Run the development server:
   ```bash
   npm run dev
   ```
4. Open [http://localhost:3000](http://localhost:3000) in your browser.
